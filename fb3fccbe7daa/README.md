# 田园物语：农场养成

一个纯前端的单页农场养成小游戏，无需安装依赖，直接用浏览器打开 `index.html` 即可运行。

## 玩法

- 在右侧“种子商店”先选中作物
- 点击中间空地播种
- 等待成熟后再次点击地块收获
- 在“仓库”里一键出售作物换金币
- 用金币解锁更多地块，或购买升级
- 体力不足时可以点击“休息恢复体力”

## 特色

- 本地自动存档（浏览器 localStorage）
- 等级成长与经验系统
- 每日天气效果
- 地块解锁、仓库扩容、成长加速等养成机制
- 响应式布局，手机和桌面都能玩

## 文件

- `index.html`：页面结构
- `style.css`：界面样式
- `script.js`：游戏逻辑
